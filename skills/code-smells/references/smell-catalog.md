# Smell Catalog

- **Long Method**: too many decisions or abstraction levels in one routine.
- **Large Class**: too many responsibilities gathered into one type.
- **Feature Envy**: behavior lives far from the data it uses most.
- **Shotgun Surgery**: one change requires many scattered edits.
- **Primitive Obsession**: raw strings, ints, and flags hide domain meaning.
- **Duplicate Logic**: same rule appears in multiple places.
